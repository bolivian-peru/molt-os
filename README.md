# osModa

[![CI](https://github.com/bolivian-peru/os-moda/actions/workflows/ci.yml/badge.svg)](https://github.com/bolivian-peru/os-moda/actions)

**osModa** is a lightweight, self‑healing operating system for AI agents. It provides encrypted peer‑to‑peer mesh networking, autonomous service discovery, and persistent SQLite‑backed rooms.
